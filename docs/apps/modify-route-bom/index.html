Modify Route/BoM

Add/Delete routes and BoMs to/from your work orders.

Flow Diagram​

Screens​

Work Order Assembly Selection​

On this screen you need to select the work order position/assembly you want add/delete routes and BoMs to/from.

Click the button to open the Work Order Assembly Search modal.

Work Order Assembly Search Modal

On this modal you need to select one of the work order assemblies listed.

You can select any item by clicking on it, which will close the modal and take you to the Operation Type Selection screen with that work order assembly already set.

You can filter the list of work order assemblies using the search box.

If you want to close the modal without making any changes, click the button.
